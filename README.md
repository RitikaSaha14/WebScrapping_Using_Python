# Python-Based Web Scraping Application for E-Commerce Data
This project is a Python-based web scraping application designed to extract, structure, and export e-commerce product data from a live website. It demonstrates a complete analytics-style workflow, covering data extraction, validation, visualization, and storage, with an interactive GUI for user-friendly access.
* The application collects product-level information such as:

* Product Name
* Price
* Rating

and makes the data available through:

* GUI-based visualization
* Console output
* CSV file export for further analysis
